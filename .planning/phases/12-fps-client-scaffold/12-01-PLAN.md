## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → Axum /fps/* | Static asset serving; no user input crosses this boundary |
| build context → Dockerfile | Copies shared/ into fps-builder; no secrets in build args |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-01-01 | Spoofing | ServeDir /fps route | accept | Static file serving has no auth surface; assets are public by design |
| T-12-01-02 | Information Disclosure | Dockerfile COPY shared/ | accept | shared/protocol.ts contains no secrets; wire protocol types are intentionally public |
| T-12-01-03 | Tampering | vite build output | accept | Build runs in CI/Docker; no runtime write access to dist/ |
</threat_model>
