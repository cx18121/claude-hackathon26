**Props interface:**
```typescript
interface DanceHudProps {
  connected: boolean
  danceScores: [number, number]
  danceBeat: { beat: number; totalBeats: number; targetPose: Array<[number, number, number, number]> } | null
}
```

**Refs (all three are needed for the beat bar logic):**
- `barRef: React.RefObject<HTMLDivElement>` — the draining fill element
- `lastBeatTimeRef: React.MutableRefObject<number>` — `useRef(0)`, tracks wall-clock time of last beat
- `beatDurationMsRef: React.MutableRefObject<number>` — `useRef(500)`, fallback until two beats arrived

**Beat bar useEffect** — depends on `[danceBeat]`:
```typescript
useEffect(() => {
  if (!danceBeat || !barRef.current) return
  const barEl = barRef.current
  const now = performance.now()
  // Compute beat interval (fallback 500ms until two events seen)
  if (lastBeatTimeRef.current > 0) {
    beatDurationMsRef.current = now - lastBeatTimeRef.current
  }
  lastBeatTimeRef.current = now
  const beatDurationMs = beatDurationMsRef.current

  // Step 1: hard snap to 100% (no transition)
  barEl.style.transition = 'width 0ms'
  barEl.style.width = '100%'
  // Step 2: force layout reflow — REQUIRED to commit snap before drain
  void barEl.offsetWidth
  // Step 3: drain linearly
  barEl.style.transition = `width ${beatDurationMs}ms linear`
  barEl.style.width = '0%'
}, [danceBeat])
```

**Render — two-row HUD band:**

Row 1 uses `.hud-names` (inherited grid: `1fr auto 1fr`, `gap: 14px`). The gap:14px is a locked
legacy value from boxing HUD — do not use it for any other element.

- Left column: `<div className="hud-p1-name"><span className="hud-label">P1</span></div>`
- Centre column (width 40%):
  ```tsx
  <div className="hud-center-name dance-beat-indicator">
    <div className="dance-beat-label">{beatLabel}</div>
    <div className="dance-beat-track">
      <div className="dance-beat-fill" ref={barRef} />
    </div>
  </div>
  ```
  where `beatLabel = danceBeat ? `${danceBeat.beat} / ${danceBeat.totalBeats}` : '— / —'`
- Right column: `<div className="hud-p2-name"><span className="hud-label">P2</span></div>`

Row 2 uses `.hud-bars dance-score-row`:

```tsx
  <span className="dance-score dance-score-p2">
    {danceScores[1].toFixed(1)}
  </span>
</div>
```
